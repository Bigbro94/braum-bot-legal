Politique de confidentialité — Braum Bot

1. Responsable du traitement

Le traitement des données est effectué par l’administrateur du bot Braum Bot, utilisé uniquement sur un serveur Discord privé dans l’Union européenne.

2. Données collectées

Braum Bot peut traiter les données suivantes :

  • Identifiant Discord (user ID)
  • Pseudo Discord
  • Identifiant Riot (puuid, nom d’invocateur)
  • Historique TFT récupéré via les API Riot (matchs, rangs, statistiques) pour affichage dans le serveur

Aucune donnée sensible n’est collectée. Aucun mot de passe n’est collecté.

3. Finalités

Les données sont utilisées pour :

  • Lier un compte Discord à un compte TFT
  • Afficher les statistiques et historiques de jeu dans le serveur privé
  • Assurer le bon fonctionnement des commandes du bot
  • Éventuellement détecter les abus du bot via des logs techniques

4. Base légale

La base légale principale est l’intérêt légitime de l’administrateur du serveur à fournir un bot de statistiques de jeu dans un cadre privé. L’utilisation du bot vaut acceptation de cet usage.

5. Destinataires

Les données peuvent être vues par :

  • Les membres autorisés du serveur Discord privé
  • Les services techniques nécessaires à l’exécution (Discord, Riot Games)

Aucune donnée n’est vendue ni partagée à des fins commerciales.

6. Durée de conservation

Les identifiants Discord/Riot sont conservés tant que l’utilisateur reste sur le serveur ou tant que l’association compte Discord ↔ Riot est nécessaire.

Les journaux techniques peuvent être conservés pour une durée plus courte.

Sur demande, les données peuvent être supprimées.

7. Droits des utilisateurs (RGPD)

Conformément au RGPD, les utilisateurs situés dans l’UE peuvent demander :

  • l’accès à leurs données,
  • la rectification,
  • la suppression,
  • la limitation du traitement.

Ces demandes doivent être adressées à l’administrateur du bot via Discord. Une vérification d’identité pourra être réalisée à partir du compte Discord.

8. Sécurité

Des mesures techniques raisonnables sont mises en place pour protéger les données. Cependant, aucune sécurité absolue ne peut être garantie.

9. Services tiers

Braum Bot utilise les services de Discord et les API de Riot Games. L’utilisation de ces services est également soumise à leurs propres conditions d’utilisation et politiques de confidentialité.

10. Modifications

La présente politique peut être mise à jour. La version publiée sur le site est la version applicable.
