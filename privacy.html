Politique de confidentialité — Braum Bot 1. Responsable du traitement Le traitement des données est effectué par l’administrateur du bot Braum Bot, utilisé uniquement sur un serveur Discord privé dans l’UE. 2. Données que nous collectons Braum Bot peut traiter les données suivantes : Identifiant Discord (user ID) Pseudo Discord Identifiant Riot (puuid, nom d’invocateur) pour associer Discord à un compte TFT Historique TFT récupéré via les API Riot (matchs, rangs, stats) pour affichage dans le serveur Aucune donnée sensible n’est collectée. Aucun mot de passe n’est collecté. 3. Finalités Lier un utilisateur Discord à son compte TFT Afficher ses statistiques/historiques dans le serveur privé Assurer le bon fonctionnement des commandes du bot Lutter contre les abus du bot (logs techniques) 4. Base légale Base légale principale : intérêt légitime de l’administrateur du serveur à fournir un bot de statistiques de jeu dans un cadre privé. Quand un utilisateur utilise le bot, il accepte que ses données de jeu soient affichées dans le serveur. 5. Destinataires Membres autorisés du serveur Discord privé Fournisseurs techniques nécessaires (Discord, Riot Games) pour exécuter les commandes Aucune revente ni partage commercial. 6. Durée de conservation Les identifiants (Discord/Riot) sont conservés tant que l’utilisateur reste sur le serveur ou tant que le bot doit maintenir l’association. Les logs techniques peuvent être conservés plus court (rotation). Sur demande, les données peuvent être effacées. 7. Droits des utilisateurs (RGPD) Dans l’UE, vous pouvez demander : accès à vos données rectification suppression limitation du traitement Pour exercer ces droits, contactez l’administrateur du bot via Discord. L’administrateur peut vérifier votre identité via votre compte Discord. 8. Hébergement et sécurité Les données sont stockées dans l’environnement choisi par l’administrateur (par ex. hébergement perso, VPS, Docker). Des mesures techniques raisonnables sont mises en place, mais la sécurité absolue ne peut être garantie. 9. Services tiers Braum Bot peut appeler les API de Riot Games et utilise Discord. L’utilisation de ces services est soumise aussi à leurs propres conditions. 10. Modifications Cette politique peut être mise à jour. La version la plus récente prévaut.
